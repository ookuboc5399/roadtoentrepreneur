# NewsAPI セットアップガイド

## 概要
このプロジェクトでは、NewsAPIを使用して実際の最新金融ニュースを取得・表示する機能を実装しています。

## NewsAPIの取得方法

1. [NewsAPI公式サイト](https://newsapi.org/) にアクセス
2. アカウントを作成（無料プランで月1,000リクエストまで利用可能）
3. APIキーを取得

## 環境変数の設定

プロジェクトのルートディレクトリに `.env.local` ファイルを作成し、以下の内容を追加してください：

```bash
# NewsAPI Configuration
NEXT_PUBLIC_NEWS_API_KEY=your_newsapi_key_here
```

`your_newsapi_key_here` を実際のNewsAPIキーに置き換えてください。

## 実装された機能

### 1. 一般ニュースAPI (`/api/external/news`)
- **金融・投資関連ニュースのみを取得（最大4件）**
- 金融、投資、株式市場、暗号通貨、経済関連のキーワードで検索
- APIキーがない場合やエラー時はモックデータを返す
- **英語ニュースを自動的に日本語に翻訳**

### 2. TradingViewスタイルニュースAPI (`/api/external/tradingview-news`)
- **金融・投資・暗号通貨関連の国際ニュースのみを取得（最大4件）**
- 金融政策、中央銀行、投資戦略に特化したクエリを使用
- 重複除去と最新順ソートを実装
- **英語ニュースを自動的に日本語に翻訳**

### 3. 翻訳機能
- MyMemory APIを使用した無料翻訳サービス
- 翻訳結果のキャッシュ機能（同じテキストの重複翻訳を防止）
- 翻訳失敗時のフォールバック機能
- 並列翻訳による高速処理

## 使用方法

### フロントエンドでの呼び出し例

```javascript
// 金融ニュースを取得（最大4件）
const response = await fetch('/api/external/news?limit=4');
const news = await response.json();

// TradingViewスタイル金融ニュースを取得（最大4件）
const tradingViewResponse = await fetch('/api/external/tradingview-news?limit=4');
const tradingViewNews = await tradingViewResponse.json();
```

## エラーハンドリング

- APIキーが設定されていない場合、自動的にモックデータを返します
- NewsAPIからの取得に失敗した場合、フォールバックとしてモックデータを返します
- 各APIは適切なエラーログを出力します

## 注意事項

- NewsAPIの無料プランには月1,000リクエストの制限があります
- MyMemory翻訳APIは無料ですが、レート制限があります
- 本番環境では適切なレート制限を実装することを推奨します
- APIキーは絶対に公開リポジトリにコミットしないでください
- 翻訳機能は初回アクセス時に時間がかかる場合があります（キャッシュにより2回目以降は高速化）

## トラブルシューティング

### よくある問題

1. **APIキーが認識されない**
   - `.env.local` ファイルが正しい場所に配置されているか確認
   - サーバーの再起動が必要な場合があります

2. **ニュースが表示されない**
   - APIキーが有効か確認
   - ネットワーク接続を確認
   - ブラウザの開発者ツールでエラーログを確認

3. **レート制限エラー**
   - NewsAPIの利用制限に達している可能性があります
   - しばらく時間を置いてから再試行してください

4. **翻訳が機能しない**
   - MyMemory APIのサービスが一時的に利用できない可能性があります
   - 翻訳失敗時は元の英語テキストが表示されます
   - ネットワーク接続を確認してください

5. **翻訳が遅い**
   - 初回翻訳時は外部APIへのリクエストのため時間がかかります
   - キャッシュ機能により、同じテキストの2回目以降は高速化されます
